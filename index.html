Understanding Permissions

Read (r / 4):

  • Files: Can view file contents.
  • Directories: Can list contents of the directory.

Write (w / 2):

  • Files: Can modify, save, or delete the file.
  • Directories: Can create, delete, or rename files within the directory.

Execute (x / 1):

  • Files: Can run the file (if it's a script or program).
  • Directories: Can "enter" or traverse into the directory.
